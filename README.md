# Project Name
> The Golden State Academy Student Records

### Installing Dependencies

From within the root directory:

```sh
cd server
npm install
npm run seed
npm start
cd ../client
npm run build
```

## Server API

### Get student list
  * GET `students`
### Get student
  * GET `students/:id`
### Add student
  * POST `students`
### Edit student
  * PUT `students/:id`
### Delete student
  * DELETE `students/:id`
